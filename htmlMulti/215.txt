   <tr>
    <td width="125" style="border-width: 1px; border-style: solid solid none;"><p><strong>&nbsp;&nbsp; 综合服务部</strong></p></td>
    <td width="160" style="border-width: 1px; border-style: solid solid none none;"><p>协助馆领导处理好全馆行政业务工作，党务工作和文化建设与交流工作。</p><p><br></p><p>主 任&nbsp; 童 琳</p><p>副主任&nbsp;<span style="text-indent: 0em;">陈咏芳&nbsp;</span></p><p><span style="text-indent: 0em;">联系电话 2291394</span></p></td>
    <td width="390" style="border-width: 1px; border-style: solid solid none none;"><p>1、协助馆领导处理好全馆行政业务工作，上传下达并组织落实学校及馆内的有关通知和会议安排，落实馆领导在文件传阅时的批示。</p><p>2、负责、安排对外联络以及来访人员的接待工作。</p><p>3、负责做好各种会议记录，各种文书的起草、保管、归档工作；做好各类统计、报表工作；</p><p>4、协助馆领导制定发展规划、工作计划、工作总结。修订各种规章制度及岗位职责等，检查执行情况，协调部门之间的工作。</p><p>5、负责印鉴的保管和使用，负责处理各种来往信函。</p><p>6、负责馆内的人事、职称、考勤等管理工作。</p><p>7、负责办公业务用品的采购、发放、登记。</p><p>8、负责本馆物资设备的购置，固定资产的管理登记、入账、物资设备报废报修。</p><p>9、负责本馆经费指标的管理，财务登记报销及各类凭证的保管。</p><p>10、负责视听室、报告厅的预约、管理工作。</p><p>11、负责图书馆宣传报道工作。</p><p>12、负责直属支部的党务工作，发布支部工作信息、传达学校通知及文件，统计党员、干部资料，撰写材料，管理、上报支部各类文件与资料。</p><p>13、负责全馆文化建设与交流工作，负责设计、策划组织系列读书文化活动及其他文化活动。</p><p>14、负责图书馆线上平台的文化宣传工作：负责图书馆主页“专题导读”、“电子展厅”及桂电微信图书馆消息的内容编辑、更新。</p><p>15、负责编辑新生培训教材及其内容更新。</p><p>16、负责图书馆布展工作，组织各类文化、艺术作品在馆内展览展示；负责馆舍环境美化工作。</p><p>17、负责管理读者协会，指导开展各类群众性读书文化活动。</p><p>18、负责全馆消防、安全、保卫、环境卫生的管理工作。</p></td>
   </tr>
   <tr>
    <td width="125" bgcolor="#dddddd" style="border-width: 1px; border-style: solid solid none;"><p><strong>&nbsp;&nbsp;&nbsp;资源建设部</strong></p></td>
    <td width="160" bgcolor="#dddddd" style="border-width: 1px; border-style: solid solid none none;"><p>负责各类电子资源和纸质资源的采购与加工，对馆藏文献加工整理，使之成为可供检索的、具有科学体系的文献集合体。</p><p><br></p><p>主 任 刘 洁<span style="text-indent: 0em;">&nbsp;</span></p><p><span style="text-indent: 0em;">联系电话 2291356</span><br></p><p><br></p></td>
    <td width="390" bgcolor="#dddddd" style="border-width: 1px; border-style: solid solid none none;"><p>1、协助馆领导制订各类型文献的采访原则、标准和规划，根据学校的学科设置、图书馆馆藏结构以及教学、科研的发展需要，负责制订图书、报刊、电子文献年度采购计划和经费预算；负责编制年度购置各类型文献及经费使用情况统计报表，进行年度工作总结。</p><p>2、 按照学校招投标工作程序，进行各类型文献的采购招投标工作；以预订、现采、邮购等各种方式和途径购置文献资源，严把文献质量关。</p><p>3、负责验收到馆的中外文图书、报刊、电子文献等，按照中图法和我馆细则对文献进行分类标引、主题标引、著录、加工和典藏，负责期刊的装订工作，负责随书光盘的压缩、上传工作。</p><p>4、负责全馆中外文图书的书目数据库和馆藏数据库的建设与维护工作。</p><p>5、负责各类型文献采购的结算、报账；负责各类型文献的验收帐、财产账的编制与管理。</p><p>6、负责赠送文献的管理登记以及寄送捐赠证书工作。</p><p>7、 负责各类资源的荐购、试用及评价。</p><p>8、负责各类资源的利用统计及分析报告。</p><p>9、负责本部门的消防安全、治安保卫和清洁卫生工作；</p></td>
   </tr>
   <tr>
    <td width="125" style="border-width: 1px; border-style: solid solid none;"><p><strong>学科与技术服务部</strong></p></td>
    <td width="160" style="border-width: 1px; border-style: solid solid none none;"><p>设计和建设图书馆网络环境，维护馆内各类硬件、软件系统，为读者进一步利用图书馆资源，提供多种形式的情报咨询、检索服务。</p><p><br></p><p>主任&nbsp;&nbsp;<span style="text-indent: 0em;">陈炎科&nbsp;</span></p><p>联系电话 2291608</p></td>
    <td width="390" style="border-width: 1px; border-style: solid solid none none;"><p>1、负责文献检索、查收查引、科技查新、负责定题(专题)服务等工作。</p><p>2、负责与各院系联系与沟通，提供学科服务，撰写学科分析报告。</p><p>3、负责馆际互借与文献传递服务、及时解答各类型的读者咨询。</p><p>4、负责组织各类专题讲座，开展读者教育及用户培训。</p><p>5、负责各种数据库使用的定期培训，负责国内外数据库培训专员的联系和培训安排。</p><p>6、负责图书馆服务器系统、计算机网络及其它各类设备的管理与维护。</p><p>7、负责图书馆网站的制作及维护，研制与开发本馆所需的各种软件。</p><p>8、负责图书馆自动化管理系统的维护和各数据库的维护与更新。</p><p>9、负责图书馆新到软硬件设备的验收、安装、维护和管理。</p><p>10、负责图书馆报告厅的管理和使用。</p><p>11、负责全馆工作人员计算机能力的培训及指导。</p><p>12、负责本部门办公场所的安全，参与治安综合治理、消防安全和清洁卫生工作。</p></td>
   </tr>
   <tr>
    <td width="125" bgcolor="#dddddd" style="border-width: 1px; border-style: solid;"><p><strong>&nbsp;&nbsp; 读者服务部</strong></p></td>
    <td width="160" bgcolor="#dddddd" style="border-width: 1px; border-style: solid solid solid none;"><p>组织和保管全馆的书籍和期刊，向读者提供各种形式的借还、阅览和图书导读等服务。</p><p><br></p><p>主任&nbsp;<span style="text-indent: 0em;">韦建敏&nbsp;</span></p><p>金鸡岭馆副主任<span style="text-indent: 0em;">田金萍&nbsp;</span></p><p><br></p><p>花江总服务台 2291357</p><p>金鸡岭服务台 2291430</p></td>
    <td width="390" bgcolor="#dddddd" style="border-width: 1px; border-style: solid solid solid none;"><p>1、负责花江校区图书馆和金鸡岭校区图书馆所有书库（包括密集书库）、阅览室和研讨室的管理工作，按时开放与闭库。</p><p>2、负责图书借还服务工作和两校区的图书互还工作。</p><p>3、负责解答来馆读者的首问咨询服务工作。</p><p>4、负责读者数据的维护，逾期费的收取，图书赔偿和读者离校手续的办理。</p><p>5、负责工作区域各种设备的使用和管理，指导读者使用各类自助设备，发现问题及时报修且跟踪直到解决。</p><p>6、负责图书与期刊的及时上架与有序排列，做好图书归架、顺架、调架、定位等。新进的图书进行清点、财产登记与签收。做好图书清点工作。</p><p>7、负责中外文期刊、报纸的验收、上架、整理和过刊过报的下架、整理、捆绑、交送装订并记录缺刊情况。</p><p>8、负责勤工助学学生的招聘、录用考核工作，指导学生协助管理图书，并按时上报学生出勤表和核对工作量。</p><p>9、负责“图书导读”栏目主题设计及定期更新和新生入馆培训工作。</p><p>10、做好图书流通业务中的各项统计工作。</p><p class="vsbcontent_end">11、负责所管辖区域的安全工作与本班次的安全巡视工作，做好安全记录。</p></td>

        function getWidth() {
            if (window.innerWidth) {
                winWidth = window.innerWidth;
            } else if ((document.body) && (document.body.clientWidth)) {
                winWidth = document.body.clientWidth;
            }
            if (document.documentElement && document.documentElement.clientHeight && document.documentElement.clientWidth) {
                winWidth = document.documentElement.clientWidth;
            }
            return winWidth;
        }

        function browserCheck() {

                            return true;
                        }
                        break;
                    case 'Safari':
                        if (isSafari) {
                            return true;
                        }
                }

            }
            return false;
